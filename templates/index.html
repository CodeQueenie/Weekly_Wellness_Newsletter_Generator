About This Project

This is an open-source demo version of the Weekly Wellness Newsletter Generator. It demonstrates the basic functionality of the application, including:

  • Fetching trending plant-based nutrition news
  • Curating healthy plant-based recipes
  • Sharing wellness tips and self-care advice
  • Generating and previewing newsletters

The full version includes additional features such as AI-generated content, advanced SEO optimization, automated scheduling, and user analytics.
